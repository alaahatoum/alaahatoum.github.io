TurtleBot Maze Challenge

This project focused on three challenging tasks using a TurtleBot:

  1. Autonomous navigation through a 4.87 m x 4.87 m maze while mapping all static objects within 8 minutes.
  2. Navigate to 5 different locations and correctly identify images using the RGB camera, comparing them to reference templates.
  3. Implement object following and stimulus-response behaviors; for example, if the robot cannot locate the user, it reacts with sound and visual feedback to simulate an emotional response.
This experience allowed us to explore ROS navigation, perception, and basic robot-human interaction.

Skills Used

Robot Operating System (ROS), C++, Robotics, Gazebo, Rviz, Deep Learning, Git, Programming

Pallet Detection & Ground Segmentation (ROS 2)

This project implements a real-time pallet detection and ground segmentation system using ROS 2 and deep learning, optimized for edge deployment on NVIDIA Jetson AGX Orin. It allows autonomous robots to detect Euro and GMA pallets, identify traversable ground, and navigate safely in dynamic warehouse and manufacturing environments.

Key challenges included integrating YOLOv8 for pallet detection and a semantic segmentation model for ground identification, building a ROS 2 node that publishes annotated camera images in real-time.

I developed testing scripts for static images and live feeds, iterated on model accuracy, and ensured system reliability for edge AI deployment. The modular package structure supports future retraining and integration into autonomous navigation systems.

Skills Used

ROS 2 (Humble), Python, YOLOv8, Semantic Segmentation, Edge AI Deployment, Deep Learning, Git, Real-Time Systems, Robotics Software Architecture, Testing & Validation

German Traffic Sign Classification CNN

Built a convolutional neural network (CNN) in PyTorch to classify German traffic signs with high accuracy. The workflow included data preprocessing, augmentation, model training on Google Colab, and evaluation using metrics like accuracy and confusion matrices. The network was designed to handle variations in lighting, angles, and partial occlusions, providing reliable predictions for real-world scenarios.

This project gave hands-on experience in deep learning, computer vision, and practical model deployment, highlighting the importance of data quality and preprocessing in achieving strong performance.

Skills Used

Python, PyTorch, CNNs, Deep Learning, Data Augmentation, Model Training & Evaluation, Image Processing, Google Colab, Computer Vision
